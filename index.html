Overview

FGLab is a machine learning dashboard, designed to facilitate performing experiments. Experiment details and results are sent to a database, which allows analytics to be performed after their completion. The server is FGLab, and the clients are FGMachines.



One creates projects on FGLab, and corresponding project implementations on FGMachine, through the use of different JSON schemas. Experiments are submitted via FGLab, and an available FGMachine client will spawn machine learning code that will conduct the experiment.



Source code, documentation and examples for FGLab and FGMachine can be found in their corresponding repositories:

  • FGLab
  • FGMachine
